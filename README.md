## Pumpai core contracts

**Here you can find all smart contracts used to build Pump.ai**

## Requirements

- Forge CLI

## Documentation

https://book.getfoundry.sh/

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
